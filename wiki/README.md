## ğŸš€ Getting Started Right Now

**Fastest way to run JoinUp:**

```bash
# Clone & navigate
git clone <repo-url>
cd JoinUp-Event-Management

# Backend (Terminal 1)
cd backend
npm install
npm run seed
npm run start:dev

# Frontend (Terminal 2)
cd frontend
npm install
npm start

# Visit http://localhost:4200
```

**Need help?** See [Quick Start Guide](./02-Quick-Start.md) or [Troubleshooting](./15-Troubleshooting.md).
