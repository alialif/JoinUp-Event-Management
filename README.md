## API Endpoints

### Authentication
- `POST /auth/register` - Register new member (with name, email, password, birthDate, gender)
- `POST /auth/login` - Login (returns JWT token)
- `GET /auth/members` - Get all members (admin only)
- `PUT /auth/members/:memberId/role` - Change user role (admin only)
- `PUT /auth/promote/:memberId` - Promote member to staff (admin only - legacy endpoint)

### Events
- `GET /events` - List all events (with categories and price)
- `POST /events` - Create event (admin/staff only, with categories and price)
- `GET /events/:id` - Get event details
- `PUT /events/:id` - Update event (admin/staff only)
- `DELETE /events/:id` - Delete event (admin only)

### Registrations
- `POST /registrations/:eventId/member/:memberId` - Register member for event
- `GET /registrations/event/:eventId` - List registrations for event

### Attendance
- `POST /attendance/mark/:eventId/member/:memberId` - Mark attendance
- `GET /attendance/event/:eventId` - List attendance for event

### Certificates
- `POST /certificates/issue/:registrationId` - Issue certificate (PDF + QR)
- `GET /certificates/verify/:registrationId?code=X` - Verify certificate

### Audit
- `GET /audit?limit=100` - Get audit logs (admin/staff only)
